# ✦ Serenity Day Spa

A premium demo website for **Serenity Day Spa** — a luxury wellness and rejuvenation spa. Built with pure HTML, CSS, and JavaScript with an embedded ElevenLabs Conversational AI agent.

## 🌸 Features

- **Hero Section** – Stunning full-screen welcome with CTAs
- **About** – Brand story with animated floating badge
- **Services** – 6 signature treatments with pricing
- **Gallery** – Spa atmosphere grid
- **Testimonials** – Client reviews in a dark elegant layout
- **Booking Form** – Interactive reservation form with confirmation
- **Contact** – Location, hours and contact details
- **Footer** – Newsletter signup and quick links
- **AI Concierge** – ElevenLabs Conversational AI widget (bottom-right)

## 🤖 ElevenLabs Integration

The site embeds an ElevenLabs conversational AI agent that allows visitors to speak or chat with the spa's virtual assistant directly from the website.

```html
<elevenlabs-convai agent-id="agent_1301kkdn42k1f3gv11yfanex8eak"></elevenlabs-convai>
<script src="https://unpkg.com/@elevenlabs/convai-widget-embed" async type="text/javascript"></script>
```

## 🛠 Tech Stack

- HTML5 (semantic)
- Vanilla CSS (custom design system, animations, responsive)
- Vanilla JavaScript (scroll effects, form handling)
- Google Fonts: Cormorant Garamond + Jost
- ElevenLabs Conversational AI Widget

## 🚀 Getting Started

Just open `index.html` in your browser — no build step required!

```bash
# Or serve locally with any static server
npx serve .
```
